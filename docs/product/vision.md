# Product Vision

Atlas is a conversation-first planning assistant that helps turn messy thought into clearer decisions and scheduled time.

## Long-term direction

- Accept messy input with near-zero friction.
- Support fluid planning conversations, not just command-style task capture.
- Turn raw thoughts into increasingly usable tasks, plans, and time commitments.
- Help the user move from reflection to execution with less overhead.
- Stay schedule-forward: when work becomes actionable, bias toward proposing or placing time on the calendar.
- Expand from simple scheduling toward more adaptive planning, guidance, and rescheduling.

## Deferred capabilities currently in view

- Adaptive task breakdown based on user capacity
- Smarter scheduling that uses fit, timing, and workload heuristics
- Richer automatic rescheduling when work slips
- Future calendar integration and broader delivery surfaces

This file should stay high-level. Concrete scope decisions belong in `mvp-requirements.md` and later roadmap docs.
